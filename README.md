# 🤖 ML Trading Bot v1.0 (EURUSD)

Професійна система для алгоритмічної торгівлі на ринку Forex (MT5), що використовує машинне навчання (**XGBoost**) для прогнозування руху ціни на основі синхронізованих даних таймфреймів **M1** та **M5**.

## 🎯 Основні характеристики (Baseline)

* **Інструмент:** EURUSD
* **Таймфрейми:** M1 (точка входу) + M5 (контекст тренду)
* **Горизонт прогнозу:** 20 хвилин
* **Модель:** XGBoost (Classifier)
* **Точність (Accuracy):** ~61% (на Out-of-Sample тестах)
* **Profit Factor:** 2.42 (з урахуванням спреду 1.2 pts)

## 🚀 Швидкий старт

### 1. Підготовка (Вихідні)

Переконайтеся, що в `config.yaml` вказано правильні шлях до MetaTrader 5 та логін.
Створіть необхідні папки:

```bash
mkdir data models logs

```

### 2. Збір даних та Навчання

Запустіть MetaTrader 5 із встановленим скриптом `PythonBridge.mq5`. Потім виконайте:

```bash
# Збір свіжих даних
python3 main.py 

# Навчання моделі
python3 train.py

```

*Очікуйте Accuracy > 0.52 та ROC-AUC > 0.55.*

### 3. Перевірка стратегії (Backtest)

Перевірте, як модель працювала б на історичних даних із врахуванням спреду:

```bash
python3 run_backtest.py

```

### 4. Запуск у режимі Paper Trading (Live)

Для спостереження за роботою бота в реальному часі без ризику для капіталу:

```bash
python3 main.py

```

Бот буде логувати всі сигнали, віртуальні прибутки та збитки у файл `logs/signals.log`.

## ⚙️ Налаштування (config.yaml)

| Параметр | Опис | Значення за замовчуванням |
| --- | --- | --- |
| `horizon` | Час утримання позиції | 20 хв |
| `threshold` | Поріг впевненості моделі | 0.58 (58%) |
| `sl_points` | Аварійний Stop Loss | 150 пунктів |
| `spread` | Спред брокера для тестів | 0.00012 (1.2 pts) |
| `risk_per_trade` | Ризик на одну угоду | 0.01 (1%) |

## 📊 Метрики та Моніторинг

Бот використовує розширене логування:

* **`bot_core.log`**: Технічний стан сокетів та обробки даних.
* **`signals.log`**: Журнал торгових операцій. Приклад:
* `🟢 BUY | Prob: 0.72 | Price: 1.08540`
* `✅ WIN (TIMEOUT) | Profit: $12.50 | Balance: $1012.50`
